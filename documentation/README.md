## Diagramas del Proyecto (Mermaid)

Para una mejor comprensión del proyecto, se han creado varios diagramas utilizando la sintaxis Mermaid. Estos diagramas visualizan diferentes aspectos del sistema Iris:

-   **Contexto del Sistema**: Proporciona una vista de alto nivel de los actores principales y los componentes del sistema Iris, explicando su propósito general.
    -   Ver: [Contexto del Sistema](Contexto%20del%20Sistema.md)

-   **Arquitectura del Sistema**: Detalla la estructura interna del sistema, incluyendo los servicios, sus interacciones, las bases de datos utilizadas (PostgreSQL y Redis), y las tecnologías clave.
    -   Ver: [Arquitectura](Arquitectura.md)

-   **Esquema de la Base de Datos**: Muestra el diagrama Entidad-Relación (ER) de la base de datos PostgreSQL, definiendo las tablas, sus columnas y las relaciones entre ellas.
    -   Ver: [Base de Datos](Base%20de%20Datos.md)

-   **Flujo de Suscripción**: Ilustra el proceso paso a paso que sigue un estudiante para suscribirse a las notificaciones del sistema.
    -   Ver: [Flujo de Suscripción](Flujo%20de%20Suscripcion.md)

-   **Flujo de Envío de Notificaciones**: Describe el ciclo completo desde que se recibe un correo electrónico de la universidad hasta que se envía una notificación de WhatsApp al estudiante.
    -   Ver: [Flujo de Envio](Flujo%20de%20Envio.md)
